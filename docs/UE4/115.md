## Angabe

Man zeichne den Graphen der Funktion $f(x)$ und bestimme alle Stellen, an denen
$f(x)$ stetig ist.

$$
\displaystyle
\left(
sgn(x):=
\begin{cases}
+1 & \text{ für } x > 0 \\
-1 & \text{ für } x < 0 \\
0 & \text{ für } x = 0 \\ 
\end{cases}
\right)
$$

$f(x) = (x-\frac{\pi}{2}) \cdot sgn(\cos{x})$
